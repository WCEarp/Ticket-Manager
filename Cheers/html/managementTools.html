Create & Edit Users!

Manage and create users from this tab!


Enter an existing user's ID in the "User ID" field and click the "Grab User" button to edit their information.


Click "Add" after creating a new user to save their information.


Click "Update" after modifying an existing user to save their information.


Click "Delete" after fetching an existing user to delete their information.


Click "Display All User" to display all current users.

Season Ticket Holder Manager!

Manage existing season ticket holders from this tab!


Enter an existing user ID in the "User ID" field.


A seat can be selected by either typing in the seat ID manually or by selecting a seat from either the Concert Hall or Playhouse by clicking their respective buttons.


When its time to notify all season ticket holders that it is time to renew their tickets, click "Notify All to Renew" to send out a reminder to all season ticket holders.

Manage Sections and Set Show Prices

From here you can pick a show to manage.


Then you can choose how to assign that show's sections and each sections price


Finally you can preview the current settings with the button above the table and then save the settings with the button below the table

Ticket Manager!

Manage the ticket sales from this tab!


Ticket IDs can be entered manually into the "Ticket ID" field or selected from a list of all sold tickets by clicking the "Display All Tickets" button.
